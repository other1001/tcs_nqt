Commonly asked interview questions

What are the four necessary conditions for deadlock?

Mutual Exclusion, Hold & Wait, No Preemption, Circular Wait

Difference between process and thread?

Process = independent, separate memory. Thread = lightweight, shares memory

What is starvation? How is it different from deadlock?

Starvation = process waits forever but system is running. Deadlock = system is stuck

Explain Round Robin scheduling with an example

Be ready to calculate average waiting time on paper

OS is theory-heavy — don't memorize. Understand with examples. If you can explain Round Robin scheduling with numbers, you'll impress most interviewers.

Commonly asked interview questions

Explain each layer of the OSI model with one example

Most asked CN question — must know this cold

When would you use TCP vs UDP?

TCP = reliability needed (file transfer, web). UDP = speed needed (video call, gaming)

What happens when you type a URL in a browser?

DNS → IP lookup → TCP handshake → HTTP request → response — know this flow

The OSI model question is almost guaranteed. Make a simple table with all 7 layers, their function, and one protocol example per layer — memorize that table.

Commonly asked interview questions

Explain all 4 OOP pillars with a real-world example

Use Car or BankAccount as your example — simple and relatable

What is the difference between Abstraction and Encapsulation?

Abstraction = hiding complexity. Encapsulation = wrapping data + methods together

Can you achieve multiple inheritance in Java? How?

Not with classes, but yes through Interfaces

What is constructor overloading?

Multiple constructors with different parameters in the same class

Pick one language (Java or Python) and be able to write small OOP examples live in the interview. Even 10 lines of code showing inheritance impresses most TCS interviewers.
